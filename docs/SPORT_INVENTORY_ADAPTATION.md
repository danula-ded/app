# Переход на магазин спортинвентаря

Если структура задания такая же, внутренние названия `Product`, `Order`, `Supplier`, `Manufacturer`, `Category`, `Unit` можно оставить. Меняются видимые тексты, файлы импорта, картинки и при необходимости номера колонок.

## Что менять

1. Название магазина и подписи страниц:
   - `core/templates/core/base.html`
   - `core/templates/core/product_list.html`
   - `core/templates/core/product_form.html`
   - `core/templates/core/product_confirm_delete.html`
   - `core/templates/core/order_list.html`
   - `core/templates/core/order_form.html`
   - `core/templates/core/order_confirm_delete.html`

2. Ресурсы:
   - логотип: `static/images/Icon.png`
   - иконка: `static/images/Icon.ico`
   - заглушка товара: `static/images/picture.png`
   - фото товаров: `media/products/`

3. Excel-файлы:
   - положить новые файлы в `core/import/`
   - открыть `core/management/commands/import_data.py`
   - поменять имена файлов в верхних переменных
   - проверить номера колонок в методах `import_products`, `import_users`, `import_orders`, `import_pickup_points`

4. Если поля остались такими же:
   - `core/models.py` не трогать
   - `core/forms.py` не трогать
   - `core/views.py` не трогать
   - менять только тексты, ресурсы и импорт

5. Если поля изменились:
   - сначала изменить `core/models.py`
   - затем `core/forms.py`
   - затем `core/views.py`
   - затем шаблоны в `core/templates/core/`
   - затем `core/management/commands/import_data.py`

## Поля товара

Текущая форма товара содержит:

- артикул
- фото
- наименование
- категория
- описание
- производитель
- поставщик
- цена
- единица измерения
- количество на складе
- действующая скидка

Для спортинвентаря обычно достаточно заменить данные в Excel: категории, производителей, поставщиков, названия и описания товаров.

## Проверка после замены

1. Запустить миграции.
2. Запустить импорт.
3. Проверить количество товаров, заказов, позиций заказов, пользователей и пунктов выдачи.
4. Открыть страницы входа, товаров, заказов, добавления и редактирования.
5. Проверить роли:
   - гость и клиент видят только список товаров
   - менеджер видит поиск, фильтр, сортировку и заказы
   - администратор видит добавление, редактирование и удаление
6. Проверить изображения:
   - товар показывает свое фото
   - без фото показывается `picture.png`
   - загруженное фото уменьшается до 300x200
