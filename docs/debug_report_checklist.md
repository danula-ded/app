# Что вставить в docx с отладкой

В задании модуля 2 просят документ `.docx` со скриншотами корректной работы системы.

Минимальный набор скриншотов:

1. Окно входа.
2. Вход гостем и список товаров.
3. Вход под клиентом и список товаров без поиска/фильтрации.
4. Вход под менеджером: видны поиск, фильтр, сортировка и кнопка `Заказы`.
5. Вход под администратором: видны кнопки добавления/редактирования/удаления.
6. Страница заказов.
7. Форма добавления товара.
8. Форма добавления заказа.
9. Ошибка при попытке удалить товар, который есть в заказе.

На экзамене проще всего:

```text
Print Screen -> Word/LibreOffice Writer -> вставить -> сохранить как docx
```

Файл можно назвать:

```text
docs/debug_report.docx
```
