# Сюда класть файлы задания

В эту папку положи все, что выдадут на экзамене:

- текст задания: `.txt`, `.md`, `.docx`, `.pdf`;
- Excel-файлы: `.xlsx`;
- CSV-файлы: `.csv`;
- дополнительные инструкции.

Картинки можно положить сюда же, но скрипт не будет читать их как текст. Он просто увидит, что файлы существуют.
